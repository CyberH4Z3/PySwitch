About PySwitch Emulator


This emulates the PySwitch open source MIDI controller firmware in your browser. You can:

  • Load configuration files from any connected controller running PySwitch (like a MIDI Captain) via MIDI into the emulator
  • Edit the configuration in a user friendly way with a Graphical User Interface
  • Save updated configuration files directly to the connected controller via MIDI

It would be just another config tool if you could not do this, too:

  • Control any connected client device (like Kemper Profiler Player) in realtime from your browser while editing the patches. The emulated device will control your Kemper just like the real one, even without any controller connected!
  • Quickly open and test all examples delivered with PySwitch, connected to your real Kemper device, all within the browser.
  • Scan for connected controllers and clients automatically
  • Edit the code in sync with the graphical editor: You can use your own implementations, they will not be destroyed by the code generator.

When you start up the application the first time, it will automatically scan all available MIDI ports for connected controllers. If one is found, it will automatically load its configuration.

After a connection has been loaded, the application scans for available clients to control. The first one found (for example a Kemper Profiler Player) will be connected, or, if none has been found, a virtual Kemper Player is used. You can override this by clicking on the button at the bottom right and selecting a device manually.
